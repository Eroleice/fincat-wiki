# 学习任务
- 计算并理解即期和远期的利率报价和价差，描述影响价差的因素；

# 基本概念
- 报价模式：P/B Price currency / Base currency，例如USD/EUR
- Spot exchange rate: 即期汇率（T+2）
- Forward exchange rate: 远期汇率
- Bid Price: 做市商以该价格买入
- Offer Price (Ask Price): 做市商以该价格卖出
- spread: 价差，通常单位用 pips，即0.0001
