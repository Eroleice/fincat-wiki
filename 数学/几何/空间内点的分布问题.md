# 问题
在一个欧几里得空间内已知某点P，求空间内最多可以有多少个点K（K~1~，K~2~ ... K~n~），使得每一个K~n~点到其他K点的距离均大于K~n~P?

# 解题思路
先简化为二维平面问题，已知某点P，求平面内有多少个点......
要求K~m~K~n~ > K~m~P，可以先考虑找到K~m~K~n~ = K~m~P的情况，也就是正三角形，所以问题转化成了：在一个圆形内，可以放下多少个以圆心为顶点的正三角形？
